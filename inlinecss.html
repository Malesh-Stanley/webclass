Cascading Style Sheet(CSS)

Cascading Style Sheet (CSS) is used to style an html document
It can be applied in three different ways as shown below

  1. Inline css
  2. Internal css
  3. External css

Advantages of CSS

  • Changing the color of elements
  • Adjusting the size and dimension of an element
  • Adjusting the layout of an element
